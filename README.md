# LoLCoachingVideoTool

LoLのコーチング動画を作成するためのElectronアプリケーション（React + TypeScript）です。
録画機能、ズーム、描画機能などを備え、スムーズなクリップ解説動画の制作を支援します。

## 📥 インストールと起動

1. **ダウンロード**: [Releasesページ](https://github.com/Giepie334/LoLCoachingVideoTool/releases) から最新のインストーラー（`LoLCoachingVideoTool-Setup-x.x.x.exe`）をダウンロードしてください。
2. **インストール**: ダウンロードしたファイルを実行します。
   - ※Windowsの保護警告が出た場合は「詳細情報」→「実行」を選択してください（署名がないため表示されます）。
3. **起動**: デスクトップに作成されたショートカットまたはスタートメニューから起動してください。

## 🚀 主な機能と使い方

### 1. 動画の読み込み
- 画面中央の「ファイルを選択」ボタン、または画面上部の入力欄から動画ファイル（mp4, webmなど）を読み込みます。

### 2. 画面操作（ズーム・移動）
- **ズーム**: マウスホイールで拡大・縮小ができます。
- **移動（パン）**: 右クリックまたは中クリック（ホイール押し込み）しながらドラッグすると画面を移動できます。
- **リセット**: `R` キー、または「R」ボタンで初期位置に戻ります。
- **ミニマップ**: `M` キー、または「M」ボタンで全体図（ミニマップ）の表示切り替えが可能です。

### 3. 描画機能 (Drawing)
- **線の描画**: 左クリックでドラッグすると線を描けます。
- **色の変更**: ツールバーの色付き丸ボタンをクリックして変更します（赤、緑、黄、水色、白）。
- **モード切替**:
    - **消えるモード**: 描いた線が3秒後に自動的に消えます（解説中に便利）。
    - **残るモード**: 描いた線がそのまま残ります。
- **クリア**: `C` キー、または「Clear」ボタンですべての線を消去します。
- **ペンの太さ**: 「⚙️ 設定」パネルから調整可能です。
- **レーザー**: 「Laser」ボタンでマウスカーソルを赤いレーザーポインターに変更できます。

### 4. 動画再生コントロール
- **再生/一時停止**: `Space` キー、または「▶」ボタン。
- **シーク**: 下部のプログレスバーを操作、または `←` / `→` キーで5秒スキップ。
- **コマ送り**: `<` / `>` (カンマ/ピリオド) キーで1フレームずつ送れます。
- **再生速度**: 「x1.0」などのドロップダウンから 0.25倍 ～ 2.0倍速まで変更可能。

### 5. 録画機能 (Recording)
- **録画開始**: 「● REC」ボタンを押すと録画を開始します。
    - **録画範囲**: アプリの画面（黒帯部分を除く映像エリア）がそのまま録画されます。ズームや描画も反映されます。
- **音声設定**: 「⚙️ 設定」パネルから調整できます。
    - **マイク選択**: 使用するマイクを選べます。
    - **音量バランス**: マイク音量とゲーム音量を個別に調整可能。
    - **音声分離**: 「音声を分離する」にチェックを入れると、ゲーム音（左）とマイク音（右）に分けて保存できます（編集ソフトでの調整用）。
    - **別保存**: 「マイク音声を別ファイルで保存」にチェックを入れると、動画ファイルとは別に音声ファイル（wav/webm）が生成されます（推奨設定）。

## ⌨️ ショートカットキー一覧
| キー | 機能 |
| :--- | :--- |
| **Space** | 再生 / 一時停止 |
| **← / →** | 5秒戻る / 進む |
| **< / >** | コマ送り (戻る / 進む) |
| **R** | ズームリセット |
| **M** | ミニマップ表示切替 |
| **C** | 描画クリア |
